Neopixels

How to code Neopixels for a BBC micro:bit using MicroPython!

What is a BBC micro:bit?

A micro:bit is a microcontroller - a tiny computer that you can program to do all sorts of cool stuff.

It has loads of built-in features, such as an LED display, an accelerometer and a radio chip!

For more information about what the micro:bit can do, click here.


Creating your first program

Our preferred editor is Mu, which is installed on our laptops - click the red snake icon from the taskbar!

You can also use the online editor, which is very similar, but our instructions below assume you are using Mu.

Add the following code into your editor (replacing any that is already there):

from microbit import *
import neopixel

display.show(Image.HAPPY)
sleep(1000)
display.scroll('Hello')
  • import adds necessary libraries to your code. In this case microbit (needed now) and neopixel (needed later).
  • display does things with the micro:bit's screen.
  • sleep adds a pause into your code - it will wait for the give amount of time before moving on to the next statement.

Connect your micro:bit to your computer, and click Flash to send your code to your micro:bit.

Does the code run on your micro:bit?

  • If your computer is showing a warning that it did not flash correctly - unplug your micro:bit, plug it in again, and try clicking Flash again!
  • If you are getting an unexpected scrolling message on the micro:bit, this is an error message! Look carefully at your code and check for typing mistakes (spelling matters, as do CAPITAL letters!).

Change the code! Try other images and changing the message. How about adding more than one image?


Setting up the Neopixels

Neopixels is a brand name for 'addressable RGB LEDs'. You can connect lots of them in a strings, and program them individually, or as a group.

Add the following code below your current code, to set up the Neopixels:

lights = neopixel.NeoPixel(pin16, 8)

red = (50, 0, 0)
orange = (50, 25, 0)
yellow = (40, 50, 0)
green = (0, 50, 0)
blue = (0, 0, 50)
indigo = (50, 0, 25)
violet = (50, 0, 50)
white = (50, 50, 50)

colours = [red, orange, yellow, green, blue, indigo, violet, white]
  • lights creates a variable to set up the Neopixels. It uses the pin on the micro:bit that you have physically attached the Neopixels to (pin16), and the number of Neopixels you have in the set (8).
  • red sets a variable to the RGB colour value that will make the colour red.
  • colours creates a list of all of the different colour variables defined above.
